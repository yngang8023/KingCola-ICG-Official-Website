# 农云时代——基于农作物病虫害检测车的一体化协作平台

“农云时代”是一款面向大棚场景的农业智能辅助平台，以树莓派 4B 和 STM32 单片机为硬件基础，利用嵌入式开发技术构建感知层，在树莓派上部署 YOLOv5 与 CBAM 模型作为图像处理层，并以 Web 端“农云时代”管理平台作为应用层，形成从环境采集、病虫害识别到种植决策支持的完整协作闭环。

项目以农业小车作为农户的“第三只眼”，针对农户在病虫害识别、防治方法选择、农药种类与用量判断等方面的现实痛点提供技术支撑。同时，平台还结合溯源二维码体系，将农产品生产数据转化为农户的“品牌名片”，既帮助种植管理，也增强农产品的可追溯性与市场信任度。

## 核心能力

- 巡逻与采集：小车具备循迹行驶、自动行驶与避障能力，可在大棚内持续巡逻
- 环境感知：通过传感器与 USB 摄像头采集环境参数和作物图像数据
- 智能诊断：在树莓派上部署 YOLOv5 + CBAM 模型识别病虫害并提升检测鲁棒性
- 图像增强：结合 CLAHE、Gamma 变换与 OpenCV 去模糊算法提升图像清晰度
- 远程监控：基于 MJPG-Streamer 视频流在 Web 端实时查看作物状态
- 决策支持：农户可根据当前环境与病虫害信息向 ChatGLM 咨询科学种植建议
- 溯源与经营：支持订单管理、信息管理与农产品溯源二维码生成

## 技术实现

- 感知与控制：树莓派 4B、STM32、环境传感器、USB 摄像头
- 通信与上传：MQTT 协议、OSS Python SDK
- 图像与算法：YOLOv5、CBAM、CLAHE、Gamma 变换、OpenCV
- 应用平台：Web 管理端、视频流监控、数据可视化、订单与溯源管理

## 项目价值

“农云时代”把智能检测、可视化监控、种植咨询、经营管理和品牌溯源整合到同一平台中，不仅帮助农户更科学地种植和防治，也提升了农产品数字化管理与市场传播能力，为现代设施农业提供了完整而实用的协作方案。
