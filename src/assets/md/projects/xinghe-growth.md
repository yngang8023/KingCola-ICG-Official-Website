# 星禾

在信息爆炸和生活节奏不断加快的当下，越来越多的人开始关注自我认知、情绪管理、职业发展与长期成长路径。但现实中，很多人在面对挑战和压力时缺乏有效的自我调节能力，也常常因为成长信息碎片化、工具质量参差不齐、跨阶段交流机会不足，而在个人成长过程中产生迷茫与孤立感。

针对这些问题，我们推出了一款专注于个人成长的 APP“星禾”。“星禾”以积极心理学理论为指导，融合 LLM 大语言模型、知识图谱等特色技术，围绕自我认知、成长分析、社交支持与成长工具构建一站式个人成长平台，致力于帮助用户提升自我认知水平、增强情绪管理能力，并探索更适合自己的成长路径。

## 核心能力

- 个性化成长分析：基于用户行为数据与基础信息，生成性格认知与成长分析报告
- 多维成长工具：围绕心理、社交、性格、自信培养、挑战社恐等方向提供多种工具
- 轨迹沉淀：记录用户工具使用轨迹，帮助用户随时回顾成长历程
- 分阶段交流：将用户划分为中学生、大学生、在职三个阶段，支持跨阶段交流与经验分享
- 智能内容推荐：结合知识图谱与个体特征推荐更匹配的帖子与内容
- 正向反馈：持续提供积极情绪价值和成长激励，强化幸福感与行动力

## 功能设计

- 成长报告：LLM 个性化生成性格认知、成长分析与建议
- 成长工具箱：治愈类、学习类、培养自信类、挑战社恐类、趣味测试类工具
- 广场系统：支持发帖、点赞、评论，并提供“同阶段 / 不同阶段”双向交流机制
- 双层索引：可选择知识图谱智能推荐内容，或查看最新发布内容

## 项目价值

“星禾”通过“数据驱动的个性化分析 + 多维成长工具 + 分阶段交流平台”的组合方式，为用户提供可持续、可记录、可陪伴的成长支持体系，帮助用户更深入地了解自己、舒缓情绪、建立连接，并在长期成长中逐步形成更清晰的方向感与行动力。
