# 智途安行——多模态地面检测与可视化系统

在城市道路管理与维护中，道路缺陷（如坑洼、裂缝等）对行人和车辆的安全构成了严重威胁。传统的道路巡检方式依赖人工巡查，效率低、成本高，且难以及时发现和修复问题。随着人工智能、计算机视觉和物联网技术的发展，基于智能感知和数据分析的自动化道路检测系统逐渐成为智慧城市建设的重要方向。

智途安行旨在通过多模态数据融合、边缘计算和云端智能分析，主要开发一个检测系统，可以通过搭载此软件的每一个边缘单元或者我们自己开发的信息采集设备对道路上的道路缺陷信息进行采集，采集到的实时视频将以MP4的格式上传至云端部署的深度学习模型进行训练，训练得到的有效信息包括道路缺陷的图片、GPS定位经纬度坐标、预测结果（是坑洼或不是坑洼);我们将利用这些信息制作一个道路缺陷点分布地图，直观地展示道路缺陷的分布与严重程度，我们设计了红色、黄色、绿色三种颜色表示缺陷程度：严重缺陷、中等缺陷、轻微缺陷；每一个颜色下我们都设置了查看详情功能，使用者可以自行查看设备采集时拍摄的缺陷照片。

该系统实现高效、精准的道路缺陷检测与可视化展示，为城市管理者和道路维护部门提供实时、直观的数据支持，提高道路维护效率，保障市民出行安全。

## 核心能力

- 边缘端与采集设备实时获取道路视频数据
- 云端深度学习模型识别坑洼、裂缝等缺陷类型
- 自动关联 GPS 经纬度并形成道路缺陷点位地图
- 按严重程度分级展示（红/黄/绿），支持点位详情追溯

## 技术栈

- 管理端：Vue、JavaScript
- 后端平台：Java
- 识别模型：Python
- 采集设备与边缘端：C++

## 项目价值

通过“自动识别 + 地图可视化 + 运维决策联动”，实现道路养护从被动巡检向主动治理升级，提升城市道路安全保障能力。
