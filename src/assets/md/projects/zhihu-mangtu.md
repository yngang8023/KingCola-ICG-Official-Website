# 智护盲途——基于YOLOv5的盲道监测综合系统

随着社会发展和城市无障碍环境建设的推进，盲道作为视障群体"出行生命线"的重要性日益凸显。然而，盲道设施被车辆侵占、杂物堆放或设计不规范等现象普遍存在，严重威胁视障人士的出行安全与城市文明形象。

为破解这一社会治理难题，我们推出了“智护盲途”——基于YOLOv5的盲道监测综合系统。该系统在硬件方面通过摄像头实时监测盲道状况采集影像数据，并通过YOLOv5算法来智能识别盲道是否被占用或存在障碍物。当盲道被占用时，系统会自动发出警告提示，并在占用超时未解决的情况下，将相关图片和数据上传至管理后台，通知管理部门及时处理。

同时在软件方面采用APP应用和WEB后台管理的高效融合管理。系统APP针对不同用户群体提供了专属功能模块，对于视障人群用户会提供语音导航、障碍提醒等功能，帮助视障人士顺利通行盲道。而对于普通城市居民用户或志愿者则内置了社区反馈功能，支持用户随时举报盲道占用或违规建设问题，并通过积分奖励机制激励用户积极参与监督。同时，社区平台还提供盲人公益活动的报名入口和分享专区，鼓励居民通过活动体验盲人生活，增强公众的无障碍意识和责任感。在WEB后台页面则对摄像头传输的数据进行集中管理，以可视化图表展示盲道占用情况、居民举报信息以及用户反馈情况，帮助管理部门高效决策和执行。

该系统不仅有效解决了盲道占用和障碍物阻挡问题，也提升了视障人士的生活质量，为城市的无障碍环境建设和社会治理提供了创新解决方案，推动了社会对无障碍设施建设的关注和投入。此外，系统的数据分析功能还能帮助政府部门实时了解盲道的使用情况，优化城市公共设施规划和维护，提高城市公共资源的利用效率。

## 核心能力

- 摄像头实时监测盲道状态，自动采集图像数据
- 基于 YOLOv5 识别盲道占用、障碍物阻挡等异常情况
- 占用超时自动预警并上传后台，触发处置流程
- APP + WEB 一体化：视障辅助、群众反馈、后台可视化管理

## 功能亮点

- 视障用户：语音导航、障碍提醒
- 普通居民/志愿者：一键举报、积分激励、公益活动入口
- 管理后台：占用趋势、举报热点、处理效率可视化

## 技术栈

- 移动端：UniApp、JavaScript
- 管理端：Vue
- 后端：Java
- 视觉检测：Python

## 项目价值

系统将“问题发现”升级为“持续治理”，不仅提高盲道可用率，也提升公众无障碍意识，为城市精细化治理提供数据支撑。
